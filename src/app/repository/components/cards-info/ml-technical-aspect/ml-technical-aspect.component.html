Aspecto Técnico de ML Desde un punto de vista más técnico, se puede decir que en Aprendizaje Automático existen diferentes algoritmos. Cada algoritmo representa una estrategia diferente para aprender la función/mapping objetivo, de modo que:

"Los algoritmos de Aprendizaje Automático se describen como aprendiendo una función objetivo (F) que mapea las variables de entrada (X) (datos) a una variable de salida (Y)."

Y = F(X)
Este último puede considerarse como una tarea de aprendizaje en general, ya que se desea hacer predicciones en el futuro dadas nuevas ejemplos de entradas. El subcampo del Aprendizaje Automático que se enfoca en crear predicciones se llama "Modelado Predictivo" y es el tipo más común de Aprendizaje Automático. En este tipo de modelado, la función se aprende para hacer predicciones sobre el resultado (Y) para nuevos datos (x). Este tipo de modelado se llama modelado predictivo o análisis predictivo y el objetivo es crear la mayor precisión en las predicciones. También se puede aprender el mapeo de la función $F(x) = Y$ para obtener más información sobre los datos utilizados. Esto se llama inferencia estadística. Hablando más profundamente sobre la función objetivo F, realmente no sabemos cómo es la forma de esta función, ni es de mucho interés, es como una caja negra para nosotros, simplemente se usa directamente. Diferentes representaciones hacen diferentes suposiciones sobre la forma de la función a aprender. Por eso es importante probar un conjunto de algoritmos diferentes en un problema de Aprendizaje Automático, porque no se sabe cuál sería la mejor estimación de la estructura de la función que se está tratando de aproximar. Pero no todo es simple, ya que también hay una tasa de error (e) que es independiente de los datos de entrada (Y) y está presente en todos los algoritmos de este estilo.
Y = F(X) + e
Este error se llama irreducible porque no importa cuán bien obtengamos una estimación, la función objetivo (F) no podrá reducir el error. Entonces, la estimación de la forma de la función F siempre tendrá una tasa de error. Resumen En resumen, en esta sección vimos:
    Los algoritmos de Aprendizaje Automático trabajan para estimar el mapeo (estructura) de una función F a partir de un conjunto de información de salida (variables) (Y) dado un conjunto de información de entrada (variables) (x). Diferentes algoritmos de Aprendizaje Automático hacen diferentes suposiciones sobre la forma de la función a descubrir. Cuando no se sabe mucho sobre la forma de la función objetivo, se deben probar diferentes algoritmos para ver cuál funciona/mejora mejor. Cada función de mapeo tiene una tasa de error inherente.
